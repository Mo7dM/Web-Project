Make of Game Station

How we built Game Station

This page shows some of the references and design steps we used while creating Game Station. Each image below represents a part of our process, from initial sketches and wireframes to layout inspiration and final UI polish.



What We Created

We built Game Station to bring together everything related to discovering and understanding video games. The website includes pages for news, ratings, and a game database, all organized in a clean and modern layout. We used a bold red color for the top navigation bar to match the energetic feel of gaming websites, and added a blue gradient bar underneath to create a smooth transition and stronger visual identity.

Design Choices

Throughout the website, we used Bootstrap to structure the layout and keep everything responsive. We customized colors, shadows, rounded corners, and hover effects using our own CSS. The Bangers font was chosen for the logo to give a playful gaming look, while Montserrat was used for clean and readable text. Shadows and rounded cards were added to make the sections feel polished and professional.

Features We Developed

We used JavaScript for features like form validation and dynamic welcome messages. Tables, images, and game cards were arranged using the Bootstrap grid to ensure they stay aligned on different screen sizes. Every page was designed to feel consistent and easy to navigate.

Tools That Helped Us

During development, we used ChatGPT to fix layout issues, understand certain coding problems,text generation and generate clean solutions for HTML, CSS, and JavaScript. This helped us speed up development while learning new techniques along the way.
